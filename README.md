# JWanandroid

## 介绍
Java版Wanandroid客户端，主体使用MvpArms，利用 MVP + RxJava + Retrofit + Glide + EventBus等框架开发。

## 主要功能

- 文章：首页、知识体系、公众号、导航、项目、广场
- 搜索：热门搜索、搜索历史
- 积分：我的积分、积分排行榜
- 分享：我的分享、文章分享
- 收藏：收藏/取消收藏文章、我的收藏
- TODO：待办、已办的增删改查
- 置顶：各个文章列表点击按钮置顶、底部导航再次点击置顶
- 设置：暗黑模式、列表动画、检查更新
- 个人：登录、头像本地设置及背景模糊
